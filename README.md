# TURF — Claim Your Ground

An AI-powered urban fitness and territory capture app built for the AMD Developer Hackathon ACT II — Track 3 Unicorn.

## What is TURF?
TURF transforms your city into a battlefield. Run, walk, or cycle through real streets to capture map territories. Complete a GPS loop around any block and that land becomes yours — stamped with your profile picture, visible to everyone in real time.

## AMD AI Integration
- Fireworks AI API running on AMD hardware powers the TURF AI Coach
- Real-time voice coaching during live runs using Gemma model via Fireworks AI
- AI Route Suggester recommends optimal running routes to capture maximum territories
- AI Weekly Performance Summary generates personalized narrative reports

## Tech Stack
- Flutter (Dart) — iOS and Android
- Supabase — PostgreSQL + Realtime + Auth + Storage
- Mapbox — 3D maps, live GPS tracking, territory polygon rendering
- Fireworks AI API on AMD Developer Cloud — AI agent features
- PostGIS — geospatial polygon territory claiming

## Features
- Live GPS activity tracking (run, walk, cycle)
- Territory capture system with 200m radius fixed zones
- Organic territory claiming — run a closed loop, claim the land inside
- Profile picture stamped as territory fill pattern on map
- Real-time group runs with live friend location sharing
- Social activity feed, friends system, clubs and squads
- Weekly and monthly leaderboards (5 categories)
- Challenges system with XP rewards
- Personal fitness goals tracking
- Badge achievement system (20 badges)
- Push notifications (realtime)
- AI Pace Coach powered by Fireworks AI + Gemma on AMD hardware
